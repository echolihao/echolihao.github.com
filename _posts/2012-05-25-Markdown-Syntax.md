# 分段
******

# 列表
******
无序列表：`*`, `-`, `+`

有序列表：`1.`, `2.`, ...

在列表中使用code block，code block前需要用平常2倍的tabs或者spaces，即2 tabs或者8 spaces。

# 强调
******
符号：星号(`*`)或者下划线(`_`)。

一个符号表示斜体，两个表示粗体，三个表示粗斜体。
